SkyCanvas :Creative Weather App

SkyCanvas is a modern, responsive, and creative weather website built using HTML, CSS, and JavaScript, powered by WeatherAPI.
Users can enter any location to view real-time weather conditions, air quality index (AQI), humidity, win, speed, temperature, and more.
The UI is designed with glowing gradients, glass effects, and auto-theme adjustments based on the weather.
Features
Search Any Location
Enter city name, postal code, coordinates (lat,long), or type "auto" to auto-detect via IP.
Uses your WeatherAPI key.
Geolocation Support
One-click button to get weather using the user’s device location (requires browser permission).
Temperature Toggle
Switch between °C and °F instantly.
Creative UI
Gradient backgrounds based on weather conditions.
Glassmorphism styling.
Weather icons and AQI details.
Responsive and modern layout.
Air Quality Index
Displays AQI (US-EPA Index) for supported locations.
Fast and Responsive
Error handling for invalid locations.
Loading indicators.
Works on mobile and desktop.
Tech Stack
HTML5
CSS3
JavaScript
WeatherAPI (current.json with aqi=yes)
Installation

Open the project folder.
